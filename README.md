# Chenye Wang Homepage

This repository hosts the personal academic homepage of Chenye Wang.

## Contents

- `index.html`: homepage content and layout
- `assets/css/theme.css`: visual styles and responsive layout
- `assets/img/`: profile and institution assets
- `assets/cv/`: downloadable CV file
- `main.tex`: LaTeX source for the CV

## Local Preview

Open `index.html` directly in a browser to preview the site.

## Deployment

The site is designed for GitHub Pages and uses static HTML, CSS, and JavaScript only.
